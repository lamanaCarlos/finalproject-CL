# Guía UX/UI y Componentes – Marketplace de Arte

Este documento define los **principios de experiencia de usuario (UX)** y la **guía inicial de interfaz (UI)**, junto con los **componentes React clave**, para asegurar coherencia visual, usabilidad y escalabilidad.

---

## 1. Principios UX del Producto

1. **El arte es el protagonista**
   - Imágenes grandes y limpias
   - Interfaz neutra que no compita con la obra

2. **Claridad por rol**
   - Cada usuario ve solo lo que necesita
   - Acciones claras y contextualizadas

3. **Confianza y transparencia**
   - Información clara de precios y comisiones
   - Estados visibles (vendido, reservado, en progreso)

4. **Flujos cortos**
   - Mínimos pasos para comprar o publicar

---

## 2. Principios UI

- Diseño limpio y contemporáneo
- Uso de espacios en blanco
- Tipografía legible (sans-serif)
- Paleta de colores neutra
- Contraste alto para accesibilidad

---

## 3. Layouts Principales

### 3.1 Layout Público
- Header con navegación
- Galería central
- Footer informativo

### 3.2 Layout Usuario Autenticado
- Sidebar o menú superior
- Área principal de contenido

## 5. Estados y Feedback al Usuario

- Loaders durante peticiones
- Mensajes claros de error
- Confirmaciones visuales tras acciones

---

## 6. Internacionalización en UI

- Textos desde archivos i18n
- Evitar strings hardcodeados
- Soporte RTL futuro

---

## 7. Accesibilidad

- Navegación por teclado
- Etiquetas ARIA
- Contraste mínimo AA

---
