Prompt 1

Adopta el rol de un producto owner y con gran experiencia en el sector del arte y su comercialización. 
Objetivo: Plan completo paso a paso para la creación de un proyecto desde cero. Dicho proyecto se trata de una aplicación web donde los diferentes artistas podrán crear sus perfiles, mostrar sus obras ya sea con dibujos o fotos de las mismas y comercializarlas a través de la aplicación. 
Requerimientos: 
- Los artistas tendrán sus perfiles privados en una sección propia donde puedan administrar sus obras. 
-Existirá una sección de administradores de la aplicación por encima de los artistas. 
- Tendrá multi-idioma, como mínimo español e ingles. 
- Debera tener una sección publica donde poder ver las obras para los usuarios no artistas, pero para poder comprar deberan registrarse. 
- Requiere de un sitema de login que distinga entre usuarios compradores, usuarios artistas y usuarios administradores. 
- Se usara React.js para la parte front end, Node.js para back end y MongoDB para base de datos. 

Si necesitas preguntar algo que no haya quedado claro, hazlo.


Preguntas necesarias para continuar (muy importantes)

¿El arte será solo digital, solo físico, o ambos?

¿La plataforma gestionará envíos físicos o solo la venta?

¿Habrá curaduría (aprobación manual de obras)?

¿Se contempla comisión por venta? ¿Porcentaje?

¿El proyecto apunta a mercado local o internacional?
